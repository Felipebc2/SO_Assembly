# Divisibilidade

Utilizando estruturas condicionais, implemente o seguinte:
    if [x] for divisível por 16:
        y = 1
    else:
        y = 0

onde:
    x = r10
    y = rax

Nesta atividade, serão executados múltiplos casos de testes. Aqui está um
exemplo:

  (dados) [0x404000] = {1 dword]}
  r10 = 0x404000
