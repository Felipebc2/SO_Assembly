# Maior de 2

Escreva um programa para identificar o maior valor entre dois inteiros com
sinal. Cada valor é um inteiro de 32 bits, e o maior entre esses dois deverá
ser inserido no registrador RAX.

Nesta atividade, serão executados múltiplos casos de testes. Aqui está um
exemplo:

  (data) [0x404000] = {2 random dwords}
  rax = 0x404000
