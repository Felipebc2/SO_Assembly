# Maior de 3

Escreva um programa para identificar o maior valor entre três inteiros com
sinal. Cada valor é um inteiro de 64 bits, e o maior entre esses dois deverá ser
inserido no registrador RAX.

Nesta atividade, serão executados múltiplos casos de testes. Aqui está um
exemplo:

  (data) [0x404000] = {3 random qwords}
  r10 = 0x404000
