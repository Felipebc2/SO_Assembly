- Escrita de Memória

Nesta atividade, você deverá ler o valor que encontra-se na posição de memória
0x404000 para o registrador rdx. Em seguida, você deverá salvar nesta mesma
posição de memória onde você leu o valor, ele mesmo incrementado de 0x42.

Valores inicializados:
        - [0x404000] = 0x140588
